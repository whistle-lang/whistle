## CLI

### Installing

#### Git

```bash
cargo install --git https://github.com/whistle-lang/whistle
```

#### Local

```bash
git clone https://github.com/whistle-lang/whistle
cd whistle
cargo install --path cli
```

### Usage

```bash
whistle -h
```
